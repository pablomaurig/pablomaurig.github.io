## üì• Installation

Clone the repo:

```bash
git clone git@github.com:pablomaurig/pablomaurig.github.io.git
cd pablomaurig.github.io
```

Install dependencies:

```bash
pnpm install
```

Run dev server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
